# RISS School Event Hub 🏫
A real-time dashboard for school activities.

### How it works:
1. **Input:** Teachers submit events via a **Microsoft Form**.
2. **Bridge:** **Power Automate** processes the data and updates the `events.json` file.
3. **Display:** This website (GitHub Pages) fetches that data and displays it as interactive cards.

### Tech Stack:
* **HTML/Tailwind CSS** (The Design)
* **JavaScript** (The Logic/Fetch Engine)
* **GitHub Pages** (Hosting)
* **Power Automate** (Data Pipeline)
